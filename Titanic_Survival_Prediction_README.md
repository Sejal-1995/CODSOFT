# Titanic-Survival-Prediction

This project is a machine learning model that predicts whether a passenger survived the Titanic disaster or not. I used Python libraries and a Logistic Regression model to build it.

GOAL:
The main goal of the project is to understand hoe machine learning works by using real dataet and making predictions based on it.

DATASET:
The dataset contains infromation about Titanic passengers such as:
*Passenger class
*Gender
*Age
*Fare
*Family members onboard(SibSp and Parch)
*Survival Status

WHAT I DID:
*Loaded and explored the dataset
*Checked and handled missing values
*Converted the text data into numbers
*Plotted some basic graphs to understand the data
*Split the data into training and testing sets
*Trained a Logistic rgression model
*Checked the accuracy of the model

RESULT:
The model gave an accuracy of around 81% on the test data.

WHAT I LEARNED:
Through this project, I understood the basic workflow of machine learning like data cleaning, visualization, training a model and evaluating results.

TOOLS USED:

*Pandas
*Numpy
*Matplotlib
*Scikit-learn
